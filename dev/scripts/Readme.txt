To Run:
-------
Simply adjust the variables representing the location of the required directories. See below.

Run checkstyle-checkall-x, where x represents the particular laboratory/project.

Required Directores:
--------------------
Detailed in each bash file.



Script Descriptions:
--------------------
checkstyle-script
- execute checkstyle with custom check jar on a single student code submission located in SUBDIR

checkstyle-checkall
- execute the checkstyle-script bash on all student code contained within a folder
- outputs a summary file of all code tested

checkstyle-analyse
- summarises the information contained in the summary file produced from checkstyle-checkall
